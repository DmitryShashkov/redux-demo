##### Redux demo app
An application developed as a demonstration for my review.

##### Featuring:
 * full Redux experience (actions / action creators, reducers, store etc.);
 * React as V (with react-redux' connect);
 * Typescript FTW.

##### Installation:
 * install ```node``` and ```tsc```;
 * pull from repo;
 * ```npm install```
 * ```npm run build```
 * ```npm run serve```
